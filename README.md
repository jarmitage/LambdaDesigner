## Getting Started

### Prerequisites

* TouchDesigner
* Haskell (only tested with [haskellstack](https://docs.haskellstack.org/en/stable/README/))

### Installing

Clone or download [LambdaDesigner-sample](https://github.com/ulyssesp/lambdadesigner-sample).
